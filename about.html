About this project

Whether you want to create a top 10 list of your favourite novels or rate some video games, Media Masterlist has you covered. You can fill out the table, add or delete rows, edit column titles, save your progress, and export the table as a spreadsheet. This project was created for Module 1 of the Get Coding program.
